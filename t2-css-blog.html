Margin and Padding....How do they work?

What is padding?

Padding is the space between web page content and the border that surrounds it, this is good to use if you are using a fancy border and a fancy font and you notice that they both run together. It helps to keep things neat and tidy so you can still read it. This will affect the appearance of any elements used on the page

What is the margin?

The margin has more to do with the format of the page itself. Have you ever noticed when you go to a page, and the words seem to run STRAIGHT to the edges of the page? It's because the person, whomever designed and built the page, did not set the margins to the correct measurements and so everything is all out of whack, which you do not want it to be. Using a margin will NOT affect the dimensions or the size of the element used on the page.

Why is it IMPORATANT to know the difference

It saves time! Knowing the difference between the margin and padding will help you in formatting your page correctly the FIRST time. It's important to edit the CORRECT element and know which portion of it you are editing so you do not spend all that time editing the wrong thing and not getting the result you want.
